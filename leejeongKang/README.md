  <h2>5. 페이지 기능</h2>
  <h3>:one: 메인 페이지</h3>
    <li>메인 페이지 입니다.</li>
    <li>일정 시간이 지나면 이미지가 슬라이딩 됩니다.</li>
          <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(23).gif", width="1000">
    
  <h3>:two: 회원관리 시스템</h3>
      <h4>🏥회원가입 페이지</h4>
        <li>회원가입을 할 수 있는 페이지입니다.</li>
        <li>메인 페이지 상단의 회원가입 문자를 클릭하면 이동합니다.</li>
        <li>아이디 중복확인을 할 수 있습니다.</li>
        <li>회원가입이 완료되면 로그인 페이지로 자동 이동합니다.</li>
                  <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(21).gif", width="1000">
    <h4>🏥로그인 페이지</h4>
        <li>로그인을 할 수 있는 페이지 입니다.</li>
        <li>메인 페이지 상단의 로그인 문자를 클릭하면 이동합니다.</li>
        <li>아이디와 비밀번호가 일치할 경우에만 로그인이 가능합니다.</li>
        <li>제목을 클릭 시 상세 내용이 하단에 뜹니다.</li>
                  <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(22).gif", width="1000">
    <h4>🏥마이 페이지</h4>
        <li>회원 정보를 수정 할 수 있는 페이지입니다.</li>
        <li>로그인을 해야만 접근 가능합니다.</li>
        <li>메인 페이지 상단의 로그인 문자를 클릭하면 이동합니다.</li>
        <li>오직 관리자만이 접근 가능합니다.</li>
                  <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(24).gif", width="1000">

  <h3>:two: 고객의소리 시스템</h3>
    <h4>🏥고객의소리 등록 페이지</h4>
        <li>게시글을 등록 할 수 있는 페이지입니다.</li>
        <li>고객의소리 리스트 페이지에서 등록 버튼을 클릭하면 이동합니다.</li>    
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/95.gif", width="1000">
    <h4>🏥고객의소리 상세 페이지</h4>
        <li>고객의소리 정보를 읽을 수 있는 페이지입니다.</li>
        <li>고객의소리 페이지에서 제목을 클릭하면 이동합니다.</li>
        <li>댓글을 등록, 삭제 할 수 있습니다.</li>
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/96.gif", width="1000">
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(98).gif", width="1000">
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(99).gif", width="1000">        
    <h4>🏥고객의소리 수정/삭제 페이지</h4>
      <ul>
        <li>게시글을 수정/삭제 할 수 있는 페이지입니다.</li>
        <li>공지사항 상세 페이지에서 '게시글을 수정하시겠습니까?' 문구 클릭 시 이동합니다.</li>
        <li>오직 글쓴이만 접근 가능합니다.</li>
        <li>삭제 아이콘을 클릭 시 해당 게시글이 삭제됩니다.</li>
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/96.gif", width="1000">
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(97).gif", width="1000">
      </ul>     
    <h4>🏥고객의소리 리스트 페이지</h4>
        <li>고객의소리 리스트를 확인 할 수 있는 페이지입니다.</li>
        <li>메인 페이지에서 '고객의소리'를 클릭하면 이동합니다.</li>
        <li>리스트 하단에 페이지네이션이 있습니다.</li>
        <li>검색을 통해 원하는 게시글을 찾을 수 있습니다.</li>
         <img src="https://github.com/kangleejeong/PersonalProject/blob/main/leejeongKang/frontend/src/assets/ezgif.com-gif-maker%20(100).gif", width="1000">
